# SS-AdaMoE: Spatio-Spectral Adaptive Mixture of Experts

This repository contains the official PyTorch implementation of the paper: **"SS-AdaMoE: Spatio-Spectral Adaptive Mixture of Experts with Global Structural Priors for Graph Node Classification"**.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Framework: PyTorch](https://img.shields.io/badge/Framework-PyTorch-orange.svg)](https://pytorch.org/)

## 1. Installation

First, clone the repository and install the required dependencies. We recommend using a virtual environment (Conda).

```bash
# Clone the repository
git clone [https://github.com/BenhanZhao/Sparse-MoE-SAM.git](https://github.com/BenhanZhao/Sparse-MoE-SAM.git)
cd Sparse-MoE-SAM

# Install dependencies
pip install -r requirements.txt
